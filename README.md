ToDo App


## Get started

1. Clone Repo

   ```bash
   https://github.com/aliabbas772/todo-app.git
   ```
   
2. Install dependencies

   ```bash
   npm install
   ```

3. Start the app

   ```bash
   npx expo start
   ```

4. features
   dark and light theme
